## 🌟 О проекте

Проект направлен на решение проблемы разрозненности информации о достопримечательностях и маршрутах города Дубна. Веб-ресурс объединяет актуальные новости, интерактивные маршруты и элементы геймификации для привлечения молодежной аудитории к внутреннему туризму.

### ✨ Ключевые особенности
*   **🎭 Событийный туризм:** Лента новостей и афиша мероприятий с поддержкой мультимедиа.
*   **📊 Прогресс исследователя:** Геймифицированный модуль в личном кабинете, отслеживающий процент изученных локаций.
*   **🌓 Адаптивный дизайн:** Поддержка **Dark Mode** и светлой темы для комфортного использования в любое время суток.
*   **🛠️ Гибкая админка:** Удобное управление контентом через Django Admin.
*   **🚀 Облачное размещение:** Проект развернут на платформе *PythonAnywhere*.
