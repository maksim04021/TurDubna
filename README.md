🌐 ТурДубна (TurDubna)
Интерактивный веб-сайт для популяризации туристического потенциала наукограда Дубна.
Проект представляет собой современную платформу для туристов, объединяющую информационный портал, систему персональных маршрутов и элементы геймификации. Разработан в рамках индивидуального итогового проекта обучающимся 10 класса лицея имени академика В.Г. Кадышевского (pp. 1, 7).

🚀 Основные возможности
Событийный туризм: Интерактивная лента новостей и мероприятий города с поддержкой мультимедиа (pp. 5-6).
Умные маршруты: Система фильтрации туристических путей с указанием географических координат (p. 5).
Личный кабинет: Система авторизации для сохранения истории активности и избранных маршрутов (p. 6).
Геймификация: Модуль «Прогресс исследователя», который в реальном времени рассчитывает процент изученных достопримечательностей (p. 5).
Адаптивность: Поддержка тёмной и светлой тем оформления (Dark/Light Mode) (pp. 5, 9).

🛠 Технологический стек
Backend: Python 3.12, Django 4 (MVT-архитектура) (pp. 4, 8).
Frontend: HTML5, CSS3, JavaScript (AJAX), Bootstrap, шаблонизатор Jinja2 (pp. 5, 8).
Database: Реляционная БД (Django ORM) (p. 5).
Deployment: PythonAnywhere (p. 5).

📂 Структура проекта
News — модель категоризации событий и новостей (p. 5).
Route — система управления маршрутами и локациями (p. 5).
Signals — автоматизация создания профилей и управления медиа-файлами (p. 5).

🔧 Установка и запуск

Клонируйте репозиторий:
bash
git clone https://github.com
Используйте код с осторожностью.

Установите зависимости:
bash
pip install -r requirements.txt
Используйте код с осторожностью.

Выполните миграции:
bash
python manage.py migrate
Используйте код с осторожностью.

Запустите сервер:
bash
python manage.py runserver
Используйте код с осторожностью.
