🌐 ТурДубна (TurDubna)
Интерактивный веб-сайт для популяризации туристического потенциала наукограда Дубна.

Проект представляет собой современную платформу для туристов, объединяющую информационный портал, систему персональных маршрутов и элементы геймификации. Разработан в рамках индивидуального итогового проекта обучающимся 10 класса лицея имени академика В.Г. Кадышевского

🚀 Основные возможности
Событийный туризм: Интерактивная лента новостей и мероприятий города с поддержкой мультимедиа
Умные маршруты: Система фильтрации туристических путей с указанием географических координат 
Личный кабинет: Система авторизации для сохранения истории активности и избранных маршрутов
Геймификация: Модуль «Прогресс исследователя», который в реальном времени рассчитывает процент изученных достопримечательностей
Адаптивность: Поддержка тёмной и светлой тем оформления (Dark/Light Mode)

🛠 Технологический стек
Backend: Python 3.12, Django 4 (MVT-архитектура)
Frontend: HTML5, CSS3, JavaScript (AJAX), Bootstrap, шаблонизатор Jinja2
Database: Реляционная БД (Django ORM)
Deployment: PythonAnywhere

📂 Структура проекта
News — модель категоризации событий и новостей
Route — система управления маршрутами и локациями
Signals — автоматизация создания профилей и управления медиа-файлами

🔧 Установка и запуск

Клонируйте репозиторий:
bash

git clone https://github.com
Используйте код с осторожностью.

Установите зависимости:
bash

pip install -r requirements.txt
Используйте код с осторожностью.

Выполните миграции:
bash

python manage.py migrate
Используйте код с осторожностью.

Запустите сервер:
bash

python manage.py runserver
Используйте код с осторожностью.
